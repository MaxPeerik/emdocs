#### 'Caused by' exception template
Use this template for formatting of exceptions in the stacktrace. Insert the stacktrace in the description by using the {STACK_TRACE} token.<br/>
<br/>
Possible tokens:<br/>
{EXCEPTION_CLASS}<br/>
{EXCEPTION_MESSAGE}
